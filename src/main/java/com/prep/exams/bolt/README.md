# Bolt - HackerRank Exam Prep

Common problem types asked by Bolt (ride-hailing platform).

## Focus Areas
- Nearest driver / K closest points (Heap)
- Surge pricing calculation
- Ride matching / assignment (Greedy)
- Trip duration / shortest path (Graph)
- Fare calculation with distance and time
