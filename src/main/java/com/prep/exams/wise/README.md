# Wise - HackerRank Exam Prep

Common problem types asked by Wise (money transfer fintech).

## Focus Areas
- Currency conversion / exchange rate graphs
- Transaction validation and processing
- Fee calculation with tiered pricing
- Rate limiting (sliding window)
- Balance tracking with concurrent operations

## Problems

| # | Problem | Pattern | Difficulty |
|---|---------|---------|------------|
| 1 | Currency Converter | BFS / Graph | Medium |
| 2 | Transaction Validator | String / HashMap | Easy |
| 3 | Fee Calculator | Binary Search / Math | Medium |
| 4 | Rate Limiter | Sliding Window / Queue | Medium |
| 5 | Balance Tracker | HashMap / Prefix Sum | Easy |
